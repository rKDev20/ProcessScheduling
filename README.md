# ProcessScheduling
Android app which simulates basic CPU processing algorithms.

Supported algorithms :

1. First come first serve.
2. Shortest Job First (Preemptive/ Non-Preemptive).
3. Priority based (Preemptive/ Non-Preemptive).
4. Round Robin.
